Juicer Docs

juicer is a fast lightweight javascript template engine, you can use it to split the data and the html structure. it can also running via node.js as demo below.

the name of juicer

data(json) like fruits, template like water, combine them with juicer.

import juicer.js

<script type="text/javascript" src="juicer.min.js"></script>

* api

compile and render the template with given data.

juicer.to_html(tpl,data,options);

only compile the template.

var tpl=juicer.compile(tpl,options);

render the template using complied template.

var tpl=juicer.compile(tpl,options);
var html=tpl.render(data);

options

{
	cache:true/false,
	loose:false/true
}

cache means that whether to cache the compiled template.

* syntax

a. ${var}

using ${} to evaluating expressions and functions.

${data.name}
${data.name|function}

b. {@each} ... {@/each}

when you need iterate a list, you can use `each` like this.

{@each data.list as item}
	${item.prop}
	${.} //index
{@/each}

you can also custom the index name.

{@each data.list as item,index}
	${item.prop}
	${index} //index
{@/each}

c. {@if} ... {@else} ... {@/if}

you can also use if-else if you need.

{@each data.list as item,index}
	{@if index===3}
		the index is 3, the value is ${item.prop}
	{@else}
		the index is not 3, the value is ${item.prop}
	{@/if}
{@/each}

* running via node.js

var juicer=require('./juicer');
var html=juicer.to_html(tpl,data,options);

* demo

<ul>
	{@each data.list as it,index}
		<li>${it.name} (index: ${index})</li>
	{@/each}
	{@each data.blah as it}
		<li>
			num: ${it.num} <br />
			{@if it.num==3}
				{@each it.inner as it2}
					${it2.time} <br />
				{@/each}
			{@/if}
		</li>
	{@/each}
</ul>

var data={
	list:[
		{name:'guokai',show:true},
		{name:'benben',show:false},
		{name:'dierbaby',show:true},
	],
	blah:[
		{num:1},
		{num:2},
		{num:3,inner:[
			{'time':'15:00'},
			{'time':'16:00'},
			{'time':'17:00'},
			{'time':'18:00'}
		]},
		{num:4}
	]
};
